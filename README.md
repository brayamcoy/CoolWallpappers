# CoolWallpappers
Creation of a wallpaper gallery application, created with React Native, making use of the free pexels photo api and use of custom design with flatlist

## Dependences

- Babel 
- React Native
- Expo
- react-native-image-pan-zoom
- expo-splash-screen

## Init project

yarn install && yarn start
|| expo start
